# mokie

A TikTok-style infinite scroll feed powered by Reddit content.

## Stack
- **Frontend**: React 18 + TypeScript + Vite + Zustand + Tailwind
- **Backend**: Node.js + Express + Playwright scraper

## Structure
```
mokie/
  apps/
    web/   # React frontend
    api/   # Express backend + scraper
```

## Quick Start
```bash
npm install
npm run dev
```
